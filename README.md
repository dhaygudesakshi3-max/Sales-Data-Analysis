# Sales-Data-Analysis
Python + Pandas Internship Project

## Objective
Analyze sales data using Python and Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Steps Performed
1. Data cleaning
2. Removed duplicates
3. Converted date column
4. Calculated total sales
5. Analyzed monthly, product-wise and city-wise sales
6. Created visualizations

## Key Insights
- Top selling products identified
- Best performing month analyzed
- Highest revenue city determined

- SalesAnalysisProject.ipynb
